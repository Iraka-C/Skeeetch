# Skeeetch 开发

Skeeetch的设计很适合进行基于画板操作的开发！这份开发者文档将介绍Skeeetch的构成、开发的方法、和开发过程中的注意事项。

## 目录

点击某个主题进入相应章节

* [用户界面（布局、事件）]()
* 画布
* 鼠标光标
* 调色盘
* 图层管理
* 画布渲染（笔刷、图层合成）
* 历史记录
* 系统设置与设置保存
* 文件打开与保存
* 外部文件导入导出
* 在线功能
